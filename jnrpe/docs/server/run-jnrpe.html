Running JNRPE

The simplest command to run JNRPE is:

UNIX
WINDOWS
cd $JNRPE_HOME/bin
./jnrpe -c ../etc/jnrpe.ini
cd %JNRPE_HOME%\bin
jnrpe -c ..\etc\jnrpe.ini
important

The JNRPE_HOME environment variable must contain the path where JNRPE has been installed
